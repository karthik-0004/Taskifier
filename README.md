# Taskifier

A NestJS backend for task and project management.

## Prerequisites

- Node.js (v18 or higher)
- npm

## Setup

```bash
# Install dependencies
npm install

# Copy environment variables and fill them in
cp .env.example .env
```

## Running the app

```bash
# development (watch mode)
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
